ax_prog_md5sum

Synopsis

     AX_PROG_MD5SUM([ACTION-IF-NOT-FOUND])

Description

This macro find a md5sum application and set the variable md5sum to the name of the application or to no if not found if ACTION-IF-NOT-FOUND is not specified, configure fail when then application is not found.
