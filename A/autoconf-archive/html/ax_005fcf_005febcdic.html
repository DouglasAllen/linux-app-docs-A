ax_cf_ebcdic

Synopsis

     AX_CF_EBCDIC

Description

If the target character set is EBCDIC, defines variables cf_cv_ebcdic, EBCDIC, and NOT_ASCII.

I originally wrote this and submitted it to the Lynx distribution. Editorial revisions by Tom Dickey, et. al.
